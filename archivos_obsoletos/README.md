# Archivos obsoletos de POHAPP

Esta carpeta contiene archivos que ya no son necesarios para el funcionamiento principal de la aplicación POHAPP, pero se conservan como referencia histórica.

## Descripción de los archivos

### Scripts de utilidades temporales
- **explore-models.js** - Script utilizado para explorar el contenido de los modelos ONNX y joblib
- **fix-model-dimensions.js** - Script para corregir problemas con las dimensiones del modelo ONNX
- **test-api-ia.js** - Script de prueba para la API de IA sin conexión a la base de datos

### Ejemplos y clientes de prueba
- **examples/api_test_client.js** - Cliente de prueba para la API de IA
- **examples/client_api_examples.js** - Ejemplos de código para consumir la API desde aplicaciones cliente

## Nota

Estos archivos se han respaldado aquí antes de eliminarlos del proyecto principal para mantener una estructura de código más limpia.

Fecha de archivo: 6 de mayo de 2025
